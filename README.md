## 🌟 Features

- 🚀 **Random email generation** – creates temporary `@yopmail.com` addresses  
- 📡 **API simulation** – sends requests to Instagram's recovery endpoint  
- 🤖 **Telegram integration** – notifies valid accounts directly to your bot  
- 🛡️ **Educational** – learn about HTTP headers, cookies, and rate limiting  

---

## ⚠️ Important: You MUST update the tool before use!

This script contains **static cookies and headers** that will expire.  
To make it work, you must **replace them with fresh values** from your own browser.

- `requests` library
